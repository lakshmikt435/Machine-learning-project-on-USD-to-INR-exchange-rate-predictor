# Machine-learning-project-on-USD-to-INR-exchange-rate-predictor

A data-driven machine learning project that predicts the next-day USD–INR exchange rate using historical currency values, crude oil prices, and gold prices.
This project demonstrates the practical application of machine learning in financial forecasting, comparing Linear Regression and Random Forest models, and implementing recursive multi-day predictions.

🚀 PROJECT OVERVIEW

Currency exchange rates are influenced by multiple global economic indicators. In this project, the prediction of USD–INR exchange rate is performed using:
-  Historical USD–INR closing prices
-  Brent crude oil prices
-  Gold prices

The system follows a complete ML pipeline:

-  Data collection
-  Data preprocessing
-  Feature engineering
-  Model training
-  Performance evaluation
-  Recursive future forecasting

-  Pandas
-  NumPy
-  Matplotlib
